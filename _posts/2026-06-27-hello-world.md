折腾了好一阵子，这个博客终于搭起来了。

## 为什么写博客

一直觉得，**学到的东西不记下来，很快就会忘掉**。写博客不是为了给别人看，更多是给自己看的——把零散的想法整理成文字，本身就是一种思考。

之后这里会记录一些：

- 编程学习笔记
- 踩过的坑和解决方案
- 读过的书、看过的文章
- 偶尔的碎碎念

## 技术栈

这个博客基于 [Jekyll](https://jekyllrb.com/) 搭建，主题是 Persephone，部署在 GitHub Pages 上。写作体验很纯粹——用 Markdown 写好文章，推送到仓库，一切就自动完成了。

```ruby
def say_hello
  puts "你好，世界！"
end
```
